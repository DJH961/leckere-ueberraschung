# stickers/egans/

Egan stickers for **Egan Mode** — the milestone minigame in Nazan Dodge (`game.html`).

Drop funny PNG/JPG/WEBP/GIF stickers of Egan in here. During an Egan round they fly
across the screen for the player to catch (+10 points each). Transparent PNGs look best.

These are loaded **separately** from the main Nazan stickers in `../` and are only ever
fetched when Egan Mode is enabled (it ships toggled OFF). Until files are added here,
Egan Mode falls back to plain badges.
