# Favicon

Drop **one image** into this folder to set the browser tab icon (favicon) for the site.

- **Any size** works — the site automatically fits the whole image into a square and adds transparent padding (nothing gets cropped).
- **Any format** works: `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`, `.svg`, `.bmp`.
- If there are several images here, the **first one alphabetically** is used.
- If this folder has no image, the site falls back to the default 🎁 emoji icon.

Just upload the picture here via the GitHub web UI (Add file → Upload files) — nothing else to do.
For the sharpest result, a roughly square image works best, but it's not required — wide or tall images just get transparent padding to make them square.
